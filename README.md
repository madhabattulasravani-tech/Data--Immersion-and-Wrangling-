Employee Data Wrangling and Exploratory Analysis
Overview:
This project demonstrates Data Immersion and Data Wrangling using Python and Pandas.
Features:
- Dataset Loading
- Data Exploration
- Missing Value Detection
- Missing Value Treatment
- Summary Statistics
- Cleaned Dataset Export
Technologies Used:
- Python
- Pandas
- VS Code
Files:
- employee_data.csv
- task1.py
- cleaned_employee_data.csv
Outcome:
The project cleans employee salary data by replacing missing values with
the average salary andgenerates a cleaned dataset for further analysis.
